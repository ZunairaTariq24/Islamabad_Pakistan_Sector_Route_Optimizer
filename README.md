# Islamabad Pakistan Sector Route Optimizer

A pathfinding and visualization system designed to model and optimize connectivity between different sectors in Pakistan.

## Overview
This project provides a digital representation of regional sectors, enabling users to calculate and visualize the most efficient travel paths. By integrating graph theory with interactive mapping, it allows for a clear analysis of spatial connectivity and path optimization.

## Features
- **Interactive Sector Mapping:** Dynamic visual representation of regional sectors using `ipyleaflet`.
- **Shortest Path Calculation:** Utilizes Dijkstra’s algorithm to compute optimized routes between sector nodes.
- **Custom Graph Architecture:** Implements a linked-list graph model to efficiently manage sector relationships and distances.
- **Responsive Interface:** Widget-based UI for intuitive route exploration and visualization.

## Tech Stack
- **Language:** Python
- **Libraries:** `ipyleaflet`, `ipywidgets`, `heapq`

## Installation
1. Clone the repository: `git clone https://github.com/ZunairaTariq24/Data-Structures-and-Algorithms`
2. Install dependencies: `pip install ipyleaflet ipywidgets`
3. Launch the environment to explore sector connectivity.

## Project Structure
- `Graph Nodes`: Defines sectors and regional connectivity.
- `Optimization Engine`: Core Dijkstra's algorithm logic.
- `Mapping Module`: Renders sector polylines and markers.
